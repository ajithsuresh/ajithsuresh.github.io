Research Focus

Three interconnected pillars that drive my work in applied cryptography

🔐

Secure Multi-Party Computation

Designing efficient MPC protocols that allow multiple parties to jointly compute on private inputs — without any party revealing their data. Focus on practical, high-throughput constructions for real-world deployment.

Secret Sharing Garbled Circuits OT Extension 3PC / 4PC
🧠

Privacy-Preserving Machine Learning

Building MPC-based frameworks for private inference and training of neural networks. Enabling ML models to be evaluated on sensitive data without exposing the model or the data to any party.

PPML Neural Networks Fixed-Point Arith. Lookup Tables
🌐

Secure Federated Learning

Combining MPC with federated learning to achieve both privacy and robustness — defending against poisoning attacks and aggregation leakage in large-scale distributed ML systems.

Federated Learning Secure Aggregation Byzantine Robustness

Signature Works

Selected publications representing key contributions across MPC, PPML, and federated learning

USENIX Sec'21 CORE A*

ABY2.0: Improved Mixed-Protocol Secure Two-Party Computation

Arpita Patra, Thomas Schneider, Ajith Suresh, Hossein Yalame

Highly efficient 2PC framework mixing arithmetic, boolean, and Yao shares. Achieves order-of-magnitude improvements over ABY in online complexity.

Paper Code
USENIX Sec'21 CORE A*

SWIFT: Super-fast and Robust Privacy-Preserving Machine Learning

Nishat Koti, Mahak Pancholi, Arpita Patra, Ajith Suresh

First 3PC PPML framework achieving fairness and guaranteed output delivery — even against a malicious adversary — at practical performance.

Paper
NDSS'20 CORE A*

BLAZE: Blazing Fast Privacy-Preserving Machine Learning

Arpita Patra, Ajith Suresh

3PC PPML framework with record-setting online communication. Pioneered the use of function-independent preprocessing to achieve blazing fast online phase.

Paper
IEEE S&P'23 CORE A*

FLUTE: Fast and Secure Lookup Table Evaluations

Andreas Brüggemann, Robin Hundt, Thomas Schneider, Ajith Suresh, Hossein Yalame

New primitive for private evaluation of lookup tables, enabling efficient non-linear activation functions in PPML with minimal communication overhead.

Paper Code
NDSS'22 CORE A*

Tetrad: Fair and Robust 4PC Framework for Privacy-Preserving Machine Learning

Nishat Koti, Arpita Patra, Rahul Rachuri, Ajith Suresh

4-party PPML protocol achieving fairness and robustness simultaneously — the first to handle a dishonest majority while guaranteeing all parties receive correct output.

Paper
CHES'26 CORE A

WW-FL: Secure and Private Large-Scale Federated Learning

Felix Marx, Thomas Schneider, Ajith Suresh, Tobias Wehrle, Christian Weinert, Hossein Yalame

Scales MPC-based secure aggregation to large federated learning deployments, addressing both privacy and Byzantine robustness at production scale.

Paper
IEEE SaTML'24 🏆 Runner-Up Best Paper

ScionFL: Efficient and Robust Secure Quantized Aggregation

Yaniv Ben-Itzhak, Helen Möllering, Benny Pinkas, Thomas Schneider, Ajith Suresh, et al.

Combines quantized aggregation with MPC to achieve communication-efficient and Byzantine-robust federated learning — awarded Runner-Up Distinguished Paper.

Paper
Journal of Cryptology'23 CORE A*

MPClan: Protocol Suite for Privacy-Conscious Computations

Nishat Koti, Shravani Patil, Arpita Patra, Ajith Suresh

Comprehensive protocol suite unifying multiple MPC settings (honest/dishonest majority, semi-honest/malicious) under a single framework optimised for practical use.

Paper
IEEE S&P'24 CORE A*

Don't Eject the Impostor: Fast Three-Party Computation With a Known Cheater

Andreas Brüggemann, Oliver Schick, Thomas Schneider, Ajith Suresh, Hossein Yalame

Exploits the identity of a known malicious party to design asymmetric 3PC protocols that remain efficient even after cheating is detected.
